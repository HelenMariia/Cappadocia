This project is the result of educational online maraphon of YouTube channel "От 0 до 1". 
The project "Cappadocia" presents a landing page for traveling tour to Cappadocia. 
It is not hosted anywhere and has not backend part. This repository presents only front part of the page. 
The project is written on HTML with CSS and JavaScript plagins (Slick for sliders and Fancybox3 for gallery).  
All images and fonts were provided by the author of the maraphon.
